# Okane

Personal finance app, demo for module devn13-08

## Setup

### PostgreSQL

[Download](https://www.postgresql.org/download/#/) and install PostgreSQL

### EF tools

```bash
dotnet tool install --global dotnet-ef --version 9.0.12
```

### PgAdmin (For DB Visualization)

[Download](https://www.pgadmin.org/download/#/) and install.
